Risk Framework

Capital Protection First

Every position is governed by hard rules — the system cannot override them, and neither can emotion.

Dynamic Stop-Loss

SL is always set at the previous Gann level — not a fixed percentage — so it adapts to the actual market structure of each stock.

RDX Composite Filter

RDX = RSI + (ADX−20)/5. CE trades require RDX > 55; PE trades require RDX < 45. Weak momentum trades are rejected automatically.

Sustain Verification

Signals must hold for 1–3 minutes (strategy-dependent) before order placement. This eliminates wick-based false breakouts.

Market Hours Guard

Scanner only runs 09:20–14:45 IST. Positions are not taken in the first 20 minutes (high volatility) or after 14:45 (end-of-day risk).

SL Sustain Before Exit

Even a stop-loss breach is confirmed for 2 minutes before exit — guarding against stop-hunts and momentary dips that recover.

No Overrides, Ever

The system executes rules mechanically. There is no manual override mechanism — protecting against emotional decisions at critical moments.
